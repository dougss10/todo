API de gerenciar tarefas simples.

## Tecnologias

- [Spring Boot]
- [Spring Security]
- [Spring Data JPA]
- [SpringDoc OpenAPI 3]
- [Postgres]

## Práticas adotadas

- API REST
- Consultas com Spring Data JPA
- Injeção de Dependências
- Tratamento de respostas de erro
- Geração automática do Swagger com a OpenAPI 3
- Autenticação StateLess com JWT
